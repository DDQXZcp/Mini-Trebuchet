# Mini-Trebuchet
Mini Trebuchet for throwing Ping Pong balls. All models are built on **Solidworks 2021**.


### Trebuchet using Robomaster M3508 motor (trebuchet_m3508_amt102)
Motor: Robomaster M3508 Motor

Motor Driver: VESC Mini V4

Encoder: AMT102-V
